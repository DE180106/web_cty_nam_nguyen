# Database folder

Thư mục này dùng để chứa các file JSON phục vụ dựng model và seed dữ liệu cho dự án.

## Dự kiến cấu trúc

- `company.json`
- `services.json`
- `products.json`
- `contacts.json`
- `users.json`
- `posts.json`

Các file JSON trong thư mục này sẽ được dùng làm nguồn dữ liệu đầu vào để:

- dựng schema/model
- seed dữ liệu mẫu
- đối chiếu nội dung website với database
